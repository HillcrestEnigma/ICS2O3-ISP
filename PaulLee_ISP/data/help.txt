Ley learns Vim!
===============
This program teaches Vim in a form of a maze, and tests the user's knowledge in a game.

Maze of Learning
================
There are five stations to visit in the maze. Each station teaches a Vim skill. Only after you visit all stations, the user may leave the maze.

Game of Testing
===============
The game creates a limited Vim environment for the user to try out their new Vim Skills.
Your objective is to delete all unwanted characters displayed in red.
Your performance is measured by how long it takes you to remove all characters in red.

Good Luck!

Exit by typing ":q".
